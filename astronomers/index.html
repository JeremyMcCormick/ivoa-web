Information for Astronomers

In this section we provide some information for astronomers who would simply like to use VO services and tools. (If you want to know how to build a VO data service following IVOA standards, or write a VO-compatible application, you should look at the Deployers section.) The IVOA is not responsible for any of these services and tools - for more detail you should consult the various VO projects, data centres, and application web pages - but we hope the information will provide a useful starting point.
